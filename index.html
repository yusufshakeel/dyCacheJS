Getting Started

Instantiate an object

First we instantiate an object of the dyCache class.

var obj = new dyCache();
Output

Note!
Output represents the state of the cache after the execution of the last method.

Methods

set(key, value)

This will create a key in the cache and will set a value.

The value can be number or string.

obj.set('num', 10);
obj.set('str', 'Yusuf Shakeel');
Output

get(key)

This will fetch the value of a given key in the cache.

obj.get('num');
obj.get('str');
Output

length()

This will give the total number of keys in the cache.

Returns a number.

obj.length();
Output

keys()

This will give all the keys in the cache.

Returns an array.

obj.keys();
Output

del(key)

This will delete a key from the cache.

obj.del('num');
Output

exists(key)

This checks if a given key exists in the cache.

Returns boolean value: true/false.

obj.exists('num');
obj.exists('str');
Output

oSet(key, value)

This will create a key in the cache that holds an object value.

obj.oSet('user', {username: 'yusufshakeel', points: 10});
obj.oSet('awesome', { a: 10, b: 20, c: 'super'});
Output

oGet(key)

This will fetch the object value saved in the given key in the cache.

obj.oGet('user');
Output

purge()

This will remove all the keys from the cache.

obj.purge();
Output
